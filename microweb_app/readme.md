# Microweb App – Flask

Dit project is een eenvoudige **Flask microweb applicatie**, gemaakt als oefening binnen het vak **DevOps 2025–26**.

De focus van dit project ligt op:
- basis Flask-structuur
- werken met **templates** en **static files**
- correcte Git & GitHub workflow
- gebruik van `.gitignore`

> ⚠️ Dit project gebruikt **geen Docker**.
