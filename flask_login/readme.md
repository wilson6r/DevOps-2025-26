# Flask Login Application (Docker)

Dit project is een eenvoudige **Flask-gebaseerde loginapplicatie** die ik heb ontwikkeld in het kader van het vak **DevOps**.

Het doel van dit project is om:
- een Flask webapplicatie te bouwen met **HTML templates en static files**
- gebruikers te laten registreren en inloggen
- zowel **onveilige (plain text)** als **veilige (hash-based)** authenticatie te demonstreren
- de applicatie **te containeriseren met Docker**

---

## Functionaliteiten

- Homepagina (`/`)
- Loginpagina (`/login`)
- Accountpagina (`/account`)
- Signup & login (versie 1 – onveilig, plain text)
- Signup & login (versie 2 – veilig, SHA-256 hashing)
- Testscript om API-endpoints automatisch te testen
- Ondersteuning voor templates (`templates/`) en static files (`static/`)

---

## Gebruikte technologieën

- **Python 3**
- **Flask**
- **SQLite**
- **Docker**
- **HTML / CSS**
- **curl (testscript)**
